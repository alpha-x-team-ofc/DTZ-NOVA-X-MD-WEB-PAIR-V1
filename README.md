# DTZ_NOVA_XMD Session Generator

WhatsApp Session Generator created by Dulina Nethmira.

## Features
- QR Code Session Generation
- Pairing Code Session Generation
- Auto-session backup to MEGA
- Auto-join WhatsApp channel
- Modern UI with custom background

## Setup
1. Clone the repository
2. Install dependencies: `npm install`
3. Set environment variables in `.env` file
4. Start server: `npm start`

## Environment Variables
- `PORT`: Server port (default: 8000)
- `MEGA_EMAIL`: Your MEGA.nz email
- `MEGA_PASSWORD`: Your MEGA.nz password

## Usage
- Visit `/` for main menu
- Visit `/qr` for QR code session
- Visit `/pair` for pairing code session
